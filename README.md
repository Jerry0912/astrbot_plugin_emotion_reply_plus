# EmotionReplyPlus

一个独立的新插件，不改原版 `astrbot_qqemotionreply`。

## 功能

- 引用消息后使用 `/贴表情 344`：给该消息贴指定编号表情
- 使用 `/查询表情 [表情：344]`：返回这个表情的编号
- 使用 `/查询表情` 后面直接跟一个 QQ 表情：返回这个表情的编号
- 引用消息后使用 `/贴表情`：回复“不告诉我贴什么表情就让我贴，那我只能给你贴个小猪了”，并贴一个小猪表情
- 引用消息后使用 `/随机贴表情 5`：随机贴 5 个表情
- 仅支持 `aiocqhttp`

## 安装

把整个 `EmotionReplyPlus` 文件夹放到：

```text
AstrBot/data/plugins/
```

然后重启 AstrBot 或在插件管理里重载。

## 命令

### 1) 贴指定表情

```text
/贴表情 344
```

### 2) 随机贴表情

```text
/随机贴表情 5
```

### 3) 查询表情编号

```text
/查询表情 [表情：344]
```

或在命令后面直接插入一个 QQ 表情。

### 4) 不写编号

```text
/贴表情
```

## 配置项

- `default_random_num`：`/随机贴表情` 不写数量时的默认值
- `max_random_num`：随机贴表情单次上限
- `time_interval`：连续贴表情的间隔秒数
- `fallback_pig_emoji_id`：未指定表情时贴的小猪表情 ID，默认 `49`
- `open_admin_mode`：开启后不能给管理员贴表情

## 注意

- 必须先引用一条消息
- 指定表情编号必须是纯数字
- 随机贴表情使用的是内置随机表情池
